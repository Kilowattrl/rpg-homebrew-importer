# Changelog

## 1.0.0

Initial public release.

- Imports individual creations, previews, library sections, full libraries, and page documents from RPG Homebrew JSON.
- Creates Markdown notes with preserved source JSON and rendered cards.
- Supports sorted category folders and quick-add imports.
- Receives account-free direct website exports through a validated clipboard handoff.
- Uses custom CSS grids with safe wrapping for unusually long values.
- Detects clickable dice notation and records results in a dockable sidebar view.
- Includes persistent roll history and a manual dice roller.
- Supports desktop and mobile, with manual clipboard fallback where required.
